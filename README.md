# Windows Active directory management Lab #


## 🧪 Lab Objectives

### Exercise 1: Offline Domain Join
- Collect Domain Controller information
- Remove a computer from the domain
- Use `DJOIN.exe` to perform offline domain join
- Suspend domain controller
- Verify domain membership

### Exercise 2: Account Management
- Configure account lockout policy
- Simulate account lockout scenario
- Manage locked accounts using PowerShell
- Enable and disable user accounts
- Identify inactive accounts
- Configure user rights and permissions
